## PDF / 인쇄 변환 경로
- 브라우저에서 `outputs/delivery/html/*.html` 또는 `outputs/resumes/*.html` 를 열고 “Print → Save as PDF” 사용.
- 자동 변환이 필요하면 repo 루트에서 `node scripts/export-delivery-pdfs.js` 실행. (macOS의 Playwright/Chromium 필요)
